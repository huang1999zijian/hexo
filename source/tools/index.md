# 本地软件

## Mac端

- 装机实用工具
  - Keka，好用的解压工具
  - ishot，支持长截图、简单的标注，弥补系统截图的不足
  - Magnet，分屏，支持三分之一、四分之一等更多分屏方式
  - Mounty，读取NTFS格式硬盘
  - Lauchpad Manager，隐藏Lauchpad图标，主要针对Adobe部分包会在Lauchpad里生成垃圾文件
  - Mac Mouse Fix，解除鼠标加速度，跟Win一样的使用直觉
  - 24 Hour Wallpaper、Pap.er，24小时动态壁纸和海量壁纸
  - 腾讯柠檬清理、DaisyDisk、CleanMyMac、AppCleaner&Unstaller，四款清理软件
  - 超级右键Power Menu，弥补原生系统不能右键创建文件
  - Utools，常见小工具集合
  - Scudo，防火墙，隔离部分应用，阻止其网络访问
  - Clipy，开源的剪贴板历史软件
  - dropover，优化文件的拖拽
  - FCPX、iMovie，剪辑软件
  - Keycastr，显示当前按键和光标，录视频必备
  
- 笔记、学习、效率软件
  - Obsidian，个人在使用的主力软件，原因够轻量，基于本地的文件结构和md的格式，缺点是许多插件会对Markdown原生格式造成语法污染，实现功能的体验也谈不上完善，但主力Markdown依靠双链管理足够使用。
  - 熊掌记，轻量的Markdown软件，层级标签很好用，整体界面美观
  - Cubox、简悦，稍后读/剪藏软件
  - Bob，轻量的翻译软件，快捷键呼出
  - GoodNotes、Marginnote、PDF experts，配合iPad端使用，日常完成PDF阅读相关
  - Downie4，下载软件，复制链接可以提取视频、音频
  - 幕布、Xmind，大纲写作、思维导图软件
  - [Drawio](https://app.diagrams.net/)，各种流程图绘制，有网页端
  - Calibre，电子书管理、格式转换
  - 欧路词典、每日英语听力、Anki， 如其名，Anki背诵闪卡制作
- 音乐软件
  - Listen one，开源的全平台音乐搜索聚合
  - LX music，开源的全平台音乐搜索、下载站
  - Music Tag Editor Pro，个人主力使用的歌曲Meta编辑器
  - Permute3， 格式转换，但会带来音质下降
  - LyricsX，桌面、状态栏、Touchbar显示歌词，部分小众歌曲检索不到歌词
  - Apple Music，苹果从iPod时代开始建立的完善的音乐管理软件，方便管理各种Meta数据，全面的让人惊讶，和iPhone、iPad增量同步（开通Apple Music会员可以用iCloud同步，但是大量文件使用iCloud同步体验并不佳，有时候一首歌一直转圈，更推荐本地同步），推荐配合上面的工具导入mp3，自我管理歌单，虽然原始，但是好用。
- 开发者工具
  - jetbrains系列IDE
  - vscode、sublimetxt、coderunner，轻量编辑器
  - Xcode，Objective-C、Swift开发
  - VMware，虚拟机，个人用户免费
  - SecureCRT、Fineshell，SSH工具
  - Typora，文档编写，Picgo，图床
  - Github、Gitee，不解释
  - iTerm2、终端，好用的终端软件
  - Alfred，比spotlight更强大的聚焦功能
  - Postman，老牌API接口调试工具

## iOS/iPadOS端

- 个人工具主要在桌面端，移动端工具大部分是全平台工具的移动版本，在Mac端已经提到，以下只列出移动端独占
- Scanner Pro，PC不能扫描文档，备忘录、文件都可以扫描文档，第三方全能扫描王也可以，任选其一即可
